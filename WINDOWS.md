# LLCAR - Быстрый старт для Windows

Это краткое руководство по установке и использованию LLCAR на Windows.

## ⚠️ Важно

В настоящее время готовых установщиков нет в релизах GitHub.
Вы можете использовать LLCAR из исходного кода или собрать установщик самостоятельно.

## Вариант 1: Использование из исходного кода (Рекомендуется)

Это самый простой и быстрый способ начать работу с LLCAR.

### Шаг 1: Установите необходимое ПО

1. **Python 3.8 или выше**
   - Скачайте с https://www.python.org/downloads/
   - При установке отметьте "Add Python to PATH"

2. **Git** (опционально, но рекомендуется)
   - Скачайте с https://git-scm.com/download/win

3. **FFmpeg**
   - Скачайте с https://ffmpeg.org/download.html
   - Выберите "Windows builds from gyan.dev"
   - Скачайте "ffmpeg-release-essentials.zip"
   - Распакуйте в `C:\ffmpeg`
   - Добавьте `C:\ffmpeg\bin` в PATH

### Шаг 2: Скачайте LLCAR

**Вариант A: Через Git (рекомендуется)**
```cmd
git clone https://github.com/llcarn8n/LLCAR.git
cd LLCAR
```

**Вариант B: Скачать ZIP**
1. Перейдите на https://github.com/llcarn8n/LLCAR
2. Нажмите "Code" → "Download ZIP"
3. Распакуйте в удобное место
4. Откройте командную строку в этой папке

### Шаг 3: Установите зависимости

```cmd
pip install -r requirements.txt
```

### Шаг 4: Получите HuggingFace токен

Токен нужен для диаризации спикеров:

1. Зарегистрируйтесь на https://huggingface.co/
2. Перейдите в https://huggingface.co/settings/tokens
3. Создайте новый токен с правами "read"
4. Скопируйте токен

Создайте файл `.env` в папке LLCAR:
```
HF_TOKEN=ваш_токен_здесь
```

### Шаг 5: Запустите приложение

**Способ 1: Через batch-файл (проще всего)**
```cmd
run_console.bat
```

**Способ 2: Через Python**
```cmd
python main.py --interactive
```

### Готово!

Теперь вы можете обрабатывать видео и аудио файлы.

## Вариант 2: Сборка установщика Windows

Если вы хотите создать установщик Windows (.exe) самостоятельно:

### Требования:
- Python 3.8+
- PyInstaller
- Inno Setup (для создания .exe установщика)

### Шаги:

1. **Установите зависимости для сборки**
   ```cmd
   pip install -r requirements-build.txt
   ```

2. **Соберите исполняемый файл**
   ```cmd
   python build_exe.py
   ```
   Результат будет в папке `dist/llcar/`

3. **Создайте установщик (опционально)**
   - Установите Inno Setup: https://jrsoftware.org/isdl.php
   - Откройте файл `installer.iss` в Inno Setup Compiler
   - Нажмите Build → Compile
   - Готовый установщик будет в `installer_output/LLCAR_Setup_1.0.0.exe`

**Подробная инструкция:** См. [BUILD.md](BUILD.md)

## Первое использование

После запуска приложения (через `python main.py --interactive` или `run_console.bat`):

1. **Выберите опцию 1 или 2** для обработки одного файла
2. **Поместите видео/аудио** в папку `input/`
3. **Выберите файл** через встроенный браузер
4. **Настройте параметры**:
   - Язык (en/ru/zh)
   - Количество спикеров (или оставьте пустым)
   - Извлечение ключевых слов
5. **Подтвердите и дождитесь** обработки
6. **Результаты** будут в папке `output/`

### Пример обработки видео

```
1. Запустите: python main.py --interactive  (или run_console.bat)
2. Выберите: 1 (Process single video file)
3. Выберите файл из списка или укажите путь
4. Язык: ru
5. Спикеры: 2
6. Ключевые слова: yes
7. Метод: tfidf
8. Форматы: json,txt
9. Подтвердите: y
10. Дождитесь результатов
```

## Использование из командной строки

Если вы предпочитаете командную строку:

```cmd
REM Обработка видео
python main.py --video input\video.mp4 --language ru

REM Обработка аудио
python main.py --audio input\audio.wav --language en

REM Интерактивный режим
python main.py --interactive
```

### Параметры командной строки

```cmd
python main.py --help

Основные параметры:
  --video ПУТЬ          Путь к видео файлу
  --audio ПУТЬ          Путь к аудио файлу
  --interactive         Интерактивный режим
  --language ЯЗЫК       en, ru или zh
  --num-speakers N      Количество спикеров
  --formats json txt    Форматы вывода
  --device cuda/cpu     Устройство для обработки
```

## Системные требования

### Минимальные
- Windows 10 или новее (64-bit)
- 8 GB RAM
- 10 GB свободного места на диске
- Процессор с поддержкой AVX2
- FFmpeg

### Рекомендуемые
- Windows 11 (64-bit)
- 16 GB RAM
- 50 GB свободного места (для моделей)
- NVIDIA GPU с 8GB+ VRAM
- Процессор Intel i7 или AMD Ryzen 7

## Решение проблем

### Проблема: "FFmpeg not found"

**Решение:**
1. Установите FFmpeg (см. Шаг 3 выше)
2. Добавьте FFmpeg в PATH
3. Перезапустите приложение

### Проблема: "HuggingFace token required"

**Решение:**
1. Получите токен на https://huggingface.co/settings/tokens
2. Добавьте в `.env` файл
3. Перезапустите приложение

### Проблема: Медленная обработка

**Решение:**
1. Используйте GPU:
   - Установите CUDA Toolkit
   - Запустите с параметром `--device cuda`
2. Или используйте менее тяжелую модель

### Проблема: "Out of memory"

**Решение:**
1. Используйте CPU вместо GPU: `--device cpu`
2. Обрабатывайте более короткие видео
3. Закройте другие приложения

### Проблема: Низкое качество транскрибации

**Решение:**
1. Укажите правильный язык
2. Укажите количество спикеров
3. Используйте качественный аудио материал
4. Попробуйте другую модель

## Удаление приложения

### Если используется из исходного кода:
1. Просто удалите папку с LLCAR

### Если собрали установщик и установили:
1. Откройте "Параметры" → "Приложения"
2. Найдите "LLCAR Video Processing Pipeline"
3. Нажмите "Удалить"
