# LLCAR - Быстрый старт для Windows

Это краткое руководство по установке и использованию LLCAR на Windows.

## Вариант 1: Установка через инсталлятор (Рекомендуется)

### Шаг 1: Скачайте установщик

Перейдите на страницу [релизов](https://github.com/llcarn8n/LLCAR/releases) и скачайте:
- `LLCAR_Setup_1.0.0.exe` - установщик для Windows

### Шаг 2: Установите приложение

1. Запустите `LLCAR_Setup_1.0.0.exe`
2. Следуйте инструкциям мастера установки:
   - Прочитайте и примите лицензию
   - Подтвердите наличие FFmpeg (или установите после)
   - Опционально: введите HuggingFace токен
3. Дождитесь завершения установки

### Шаг 3: Установите FFmpeg (если не установлен)

FFmpeg требуется для обработки видео:

1. Скачайте FFmpeg:
   - Перейдите на https://ffmpeg.org/download.html
   - Выберите "Windows builds from gyan.dev"
   - Скачайте "ffmpeg-release-essentials.zip"

2. Установите FFmpeg:
   - Распакуйте архив (например, в `C:\ffmpeg`)
   - Добавьте в PATH: `C:\ffmpeg\bin`
   - Перезапустите компьютер

### Шаг 4: Получите HuggingFace токен

Токен нужен для диаризации спикеров:

1. Зарегистрируйтесь на https://huggingface.co/
2. Перейдите в https://huggingface.co/settings/tokens
3. Создайте новый токен с правами "read"
4. Скопируйте токен

Если вы не ввели токен при установке:
1. Откройте папку установки (обычно `C:\Program Files\LLCAR`)
2. Создайте файл `.env`
3. Добавьте строку: `HF_TOKEN=ваш_токен_здесь`

### Шаг 5: Запустите приложение

**Способ 1: Через меню Пуск**
- Найдите "LLCAR Video Processing Pipeline"
- Запустится интерактивная консоль

**Способ 2: Через ярлык на рабочем столе**
- Дважды кликните на ярлык LLCAR

**Способ 3: Через командную строку**
```cmd
llcar.exe --interactive
```

### Готово!

Теперь можете обрабатывать видео и аудио файлы.

## Вариант 2: Портативная версия (без установки)

### Шаг 1: Скачайте портативную версию

Скачайте `LLCAR_Portable.zip` из [релизов](https://github.com/llcarn8n/LLCAR/releases)

### Шаг 2: Распакуйте

1. Создайте папку (например, `C:\LLCAR`)
2. Распакуйте туда содержимое ZIP файла

### Шаг 3: Настройте

1. Установите FFmpeg (см. выше)
2. Создайте файл `.env` с HuggingFace токеном (см. выше)

### Шаг 4: Запустите

Дважды кликните на `llcar.bat`

---

## Первое использование

После запуска приложения:

1. **Выберите опцию 1 или 2** для обработки одного файла
2. **Поместите видео/аудио** в папку `input/`
3. **Выберите файл** через встроенный браузер
4. **Настройте параметры**:
   - Язык (en/ru/zh)
   - Количество спикеров (или оставьте пустым)
   - Извлечение ключевых слов
5. **Подтвердите и дождитесь** обработки
6. **Результаты** будут в папке `output/`

### Пример обработки видео

```
1. Запустите: llcar.bat
2. Выберите: 1 (Process single video file)
3. Выберите файл из списка или укажите путь
4. Язык: ru
5. Спикеры: 2
6. Ключевые слова: yes
7. Метод: tfidf
8. Форматы: json,txt
9. Подтвердите: y
10. Дождитесь результатов
```

## Использование из командной строки

Если вы предпочитаете командную строку:

```cmd
REM Обработка видео
llcar.exe --video input\video.mp4 --language ru

REM Обработка аудио
llcar.exe --audio input\audio.wav --language en

REM Пакетная обработка (через интерактивный режим)
llcar.exe --interactive
```

### Параметры командной строки

```cmd
llcar.exe --help

Основные параметры:
  --video ПУТЬ          Путь к видео файлу
  --audio ПУТЬ          Путь к аудио файлу
  --interactive         Интерактивный режим
  --language ЯЗЫК       en, ru или zh
  --num-speakers N      Количество спикеров
  --formats json txt    Форматы вывода
  --device cuda/cpu     Устройство для обработки
```

## Системные требования

### Минимальные
- Windows 10 или новее (64-bit)
- 8 GB RAM
- 10 GB свободного места на диске
- Процессор с поддержкой AVX2
- FFmpeg

### Рекомендуемые
- Windows 11 (64-bit)
- 16 GB RAM
- 50 GB свободного места (для моделей)
- NVIDIA GPU с 8GB+ VRAM
- Процессор Intel i7 или AMD Ryzen 7

## Решение проблем

### Проблема: "FFmpeg not found"

**Решение:**
1. Установите FFmpeg (см. Шаг 3 выше)
2. Добавьте FFmpeg в PATH
3. Перезапустите приложение

### Проблема: "HuggingFace token required"

**Решение:**
1. Получите токен на https://huggingface.co/settings/tokens
2. Добавьте в `.env` файл
3. Перезапустите приложение

### Проблема: Медленная обработка

**Решение:**
1. Используйте GPU:
   - Установите CUDA Toolkit
   - Запустите с параметром `--device cuda`
2. Или используйте менее тяжелую модель

### Проблема: "Out of memory"

**Решение:**
1. Используйте CPU вместо GPU: `--device cpu`
2. Обрабатывайте более короткие видео
3. Закройте другие приложения

### Проблема: Низкое качество транскрибации

**Решение:**
1. Укажите правильный язык
2. Укажите количество спикеров
3. Используйте качественный аудио материал
4. Попробуйте другую модель

## Удаление приложения

### Если установлено через инсталлятор:
1. Откройте "Параметры" → "Приложения"
2. Найдите "LLCAR Video Processing Pipeline"
3. Нажмите "Удалить"

### Если используется портативная версия:
1. Просто удалите папку с приложением
